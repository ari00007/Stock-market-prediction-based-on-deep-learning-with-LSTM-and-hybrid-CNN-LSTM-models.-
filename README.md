# Stock-market-prediction-based-on-deep-learning-with-LSTM-and-hybrid-CNN-LSTM-models.-
Deep learning-based stock price forecasting using CNN-LSTM and LSTM models on historical S&P 500 stock data, with feature engineering, time-series forecasting, and model comparison against ARIMA.
This project implements a stock price prediction system using CNN-LSTM and LSTM neural networks. Historical stock market data from the S&P 500 dataset (all_stocks_5yr.csv) is preprocessed, engineered with technical indicators, and used to train deep learning models for time-series forecasting. The repository also includes model evaluation, forecasting, and visualization workflows.

Features

Data preprocessing and normalization

Feature engineering (RSI, MACD, moving averages, volatility, returns)

CNN-LSTM and Vanilla LSTM model implementations

ARIMA baseline comparison

Performance evaluation using RMSE, MAE, R², and MAPE

Future stock price forecasting and visualization
